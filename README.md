# Evangadi Forum Project  -- June 2025 batch-Group-1 
 
## Project Overview
The Evangadi Forum project aims to create a Q&A platform for Evangadi Networks, allowing users to ask programming-related questions and receive answers from their peers. The platform will provide user registration, login, and the ability to post and answer questions.

## Objectives
- To create a user-friendly interface where Evangadi students can interact and help each other.
- To implement robust backend functionality to handle user actions and data storage.
- To ensure security and performance optimization of the application.
